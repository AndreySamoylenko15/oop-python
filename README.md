# 🛒 Product Class (Python OOP)

Простий приклад об'єктно-орієнтованого програмування (OOP) на Python.

---

## 📌 Опис

Цей проєкт демонструє:
- створення класу `Product`
- роботу з об'єктами
- методи класу
- базову логіку перевірок і змін даних

---

## 🧩 Функціонал

Клас `Product` має:

- `name` — назва товару
- `price` — ціна

### Методи:

- `show_info()` — виводить інформацію про товар
- `discount(amount)` — зменшує ціну
- `is_expensive()` — перевіряє чи товар дорогий (> 20)

---

## ▶️ Як працює програма

1. Створюються 3 товари:
   - Milk (30)
   - Bread (25)
   - Apple (10)

2. Виводяться тільки дорогі товари

3. Для одного товару застосовується знижка
